Bagaimana menggunakan fitur-fitur dari ACROSS?

Untuk menggunakan fitur, klik pilihan "Fitur" pada bagian atas laman web. Kemudian, pilih Fitur yang Anda inginkan. Pada laman tersebut akan disebutkan cara menggunakan fitur yang Anda pilih.

Bagaimana cara melihat data yang sudah pernah dianalisis?

ACROSS dilengkapi dengan penyimpanan data sementara, fitur ini memungkinkan Anda untuk melihat dan mencetak data yang pernah dianalisis. Berikut adalah langkah-langkah untuk melihat data Anda:

  1. Klik tombol fitur pada bagian atas laman web.
  2. Kemudian geser ke bawah, klik pada pilihan "Lihat data ladang Anda".
  3. Masukkan token yang telah diberikan saat data ladang Anda diunggah.
  4. Klik muat data. Sistem akan memuat data Anda untuk beberapa saat.
  5. Saat data Anda telah muncul, Anda bisa menekan tombol "Cetak" untuk mencetak data. Pastikan perangkat Anda telah terhubung dengan printer.
  6. Jika tidak ada printer, klik kanan pada browser Anda, lalu klik "Print". Saat pilihan muncul, pilih "Save as PDF", kemudian klik "Save". Data akan tersimpan ke perangkat Anda.

Aerial Crop Scanner System atau ACROSS bertujuan untuk membantu petani dalam mendeteksi dan menganalisis kesehatan tanaman padi akibat gangguan hama dan penyakit.
